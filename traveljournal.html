Travel Journal

Places I've visited

This project was a solo project at Scrimba. This meant that I only had the Figma and some description on how the app should function and from that I had to create the app basically from scratch. This is a static page that uses React, and in fact it is my first ever React website.

I am not ashamed to say that although writing the code for this app was very easy, deploying the page to Github Pages was quite difficult as it works completely differently with vanilla pages than with React apps. I used Vite to build the app.

It was fun to get to know React and build static sites with custom components. There is something fulfilling in the fact that I only need to change the data file in order to change the card components on the page.
